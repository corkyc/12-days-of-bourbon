# 12 Days of Bourbon Advent Calendar

Welcome to the "12 Days of Bourbon" Advent Calendar! Each day, you'll discover a new bourbon bottle. The calendar is interactive, with the option to track your progress and reset it if you'd like to start over.

## Features:
- **Interactive Advent Calendar:** Click on each door to reveal a new bottle of bourbon.
- **Progress Tracking:** Keep track of which days you've unlocked.
- **Reset Progress:** Option to reset the progress and start from Day 1.
- **All Bottles View:** See all 12 bourbons at once.

## Getting Started

To deploy this website on GitHub Pages, follow these steps:

1. **Clone this repository** to your local machine.
2. **Upload the files** to a GitHub repository.
3. **Enable GitHub Pages** in the repository settings.

Once deployed, you can access the site at the URL provided by GitHub Pages.
